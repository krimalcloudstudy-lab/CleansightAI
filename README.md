# Smart AI Recycle ♻️

An AI-powered waste classification and disposal guidance system.

## Features
- Location detection
- Waste image upload
- AI-based waste classification
- Map-based disposal guidance

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- AI: Google Gemini API
- Maps: Leaflet / Google Maps
- Backend: Firebase
